# 小城爆火预测系统

> 继淄博、天水、哈尔滨之后——用 AI 抢在热搜之前，预测下一座爆火的小城。

基于「五信号融合框架」的城市旅游爆火预测系统。通过多源信号融合 + AI 叙事分析 + 历史模式类比，在爆火前 2-8 周捕捉预警信号。

## 核心功能

| 功能 | 说明 |
|------|------|
| 交互式中国地图 | 15 座历史爆火城市以脉冲光点标注在真实地理位置，按爆火模式着色 |
| 点击查看详情 | 点击任意城市标记 → 右侧滑入详情面板：五信号雷达图 + 信号条 + 爆火生命周期时间线 + 分析洞察 |
| 五信号理论模型 | 左侧信息栏展示模型框架（含权重）+ 爆火生命周期四阶段 + 回测统计数据 |
| 搜索实时分析 | 搜索任意城市名 → AI 实时推演五信号 → 雷达图 + 打字机效果分析报告 + 历史类比 + 建议操作 |
| 模式筛选 | 按美食驱动/文化体验/话题造梗等 8 种模式筛选地图标记 |
| 回测验证 | 15 城 T-4 周信号回测，87% 命中率 |
| 趋势预测 | 6 座候选城市按爆火指数排序 |

## 快速启动

**无需安装任何依赖，双击 `index.html` 即可打开。**

```bash
# 或者用本地服务器
python -m http.server 8080
# 然后访问 http://localhost:8080
```

## 技术栈

- **ECharts 5.5** — 地图渲染 + 雷达图
- **DataV GeoJSON** — 中国行政区划地图数据（已内嵌）
- **纯原生 HTML/CSS/JS** — 无构建工具，无框架依赖
- **200+ 城市坐标库** — 支持任意城市搜索定位

## 五信号融合模型

| 信号 | 权重 | 说明 |
|------|------|------|
| 短视频动量 | 30% | 监测视频发布量的加速度（二阶导），非绝对量 |
| 搜索-预订不对称 | 25% | 搜索增速 >> 预订增速的剪刀差，OTA 独家壁垒 |
| 基建先行 | 15% | 新增高铁/航线/酒店的异常增长 |
| 叙事钩子 | 20% | LLM 分析是否存在可传播的「梗」 |
| 区域溢出 | 10% | 热门城市周边「平替」搜索量上升 |

详细理论依据见 [docs/methodology.md](docs/methodology.md)。

## 回测结果

- **15 座历史爆火城市**回测验证
- **87% T-4 周命中率**（13/15 城在热搜前 4 周已被预警）
- **平均提前 4.2 周**预警
- 漏报 2 城（开封·事件突发型、腾冲·季节性短窗口）已给出改进方向

## 15 座案例城市

淄博、天水、哈尔滨、开封、洛阳、西安、长沙、重庆、泉州、柳州、大理、甘孜、延吉、景德镇、腾冲

覆盖 8 种爆火模式：美食驱动、文化体验、话题造梗、人物 IP、视觉奇观、消费打卡、影视带动、季节性景观。

## 数据声明

- 历史城市的信号数据基于公开信息的回测估算
- 搜索任意城市的 AI 分析为基于哈希的确定性推演（同一城市每次结果一致），未来可替换为真实大模型 API
- 预测不构成旅游投资建议，模型价值在于缩小关注范围
